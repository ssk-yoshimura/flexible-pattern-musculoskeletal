# Data for:
Accelerating Musculoskeletal Robotics through Parametric Design and 3D-Printed Flexible Structures

## Overview

This repository contains all raw experimental data associated with the manuscript.

The data correspond to the mechanical characterization, joint validation,
transmission evaluation, and motion experiments described in the paper.

## Repository Structure

Design/  
 Design-related files and parametric structure data.

Creep/  
 Creep and recovery experimental data.

Fatigue/  
 Material fatigue test data.

SpeedDependent/  
 Rate-dependent hysteresis test data.

ParamChange/  
 Parameter variation experiments.

BowdenCable/  
 Transmission efficiency and friction evaluation data.

JointFatigue/  
 Joint fatigue experiments.

JointStressRelaxation/  
 Joint stress relaxation measurements.

Leg_experiment/  
 Leg-level mechanical validation experiments.

Motion_experiment/  
 Integrated Motion experiment data.
